A Tamagotchi-style pet simulator game built in C++ using SFML.

## Built With
- **C++20**
- **SFML 2.6.x**
- **CMake** for build system

### Build Steps

```bash
git clone https://github.com/jenice11/TamaTama.git
cd tamatama
cmake -B build -S .
cmake --build build --config Release
```
Run
```
./build/bin/Release/TamaTama
```
